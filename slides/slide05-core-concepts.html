CWL Core Concepts

Two building blocks for everything

CommandLineTool

Wraps a single command-line program

EXAMPLE

cwlVersion: v1.2
class: CommandLineTool

baseCommand: echo

inputs:
  message:
    type: string
    inputBinding:
      position: 1

outputs:
  output:
    type: stdout

Workflow

Connects multiple tools together

EXAMPLE

cwlVersion: v1.2
class: Workflow

inputs:
  input_file: File

steps:
  step1:
    run: tool1.cwl
    in:
      file: input_file
    out: [result]

outputs:
  final:
    outputSource: step1/result
